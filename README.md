# my-electron-app
Electron Quick Start example

see Quick Start https://www.electronjs.org/docs/tutorial/quick-start

# Install electronjs

    npm init -y
    npm i --save-dev electron

# Install
    npm install

# start
    npm start

# Package and deploy

    npm install --save-dev @electron-forge/cli
    npx electron-forge import
    npm run make
